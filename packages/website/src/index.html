HOW IT WORKS

Three steps. Thirty seconds.

From zero to deployed backend faster than you can write a config file.

1

Create your project

One command scaffolds your entire backend. Database, auth, storage, and API routes are all configured automatically.

$ npx create-vibekit my-app
2

Write your logic

Import what you need. Database queries, auth flows, file uploads. Everything works together with a single import.

import { db, auth } from 'vibekit'
3

Deploy globally

One command deploys everything. Frontend to the edge, backend to the cloud, database provisioned and migrated.

$ vibekit deploy
